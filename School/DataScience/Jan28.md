There is a function called `map` in SQL that will be applied to every row of another tabe to transform that data into a more or less colum rows of data.

> <u>For Example</u>
>
> If there is a table that had the column address, but it contains the address as city,street, zip all in a single column. The `map` function can be used to create three different columns to separate that data
